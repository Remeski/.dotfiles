## Remeski dotfiles

### Requirements

- a Nerdfont installed
- oh-my-zsh

### Installing

#### oh-my-zsh

- Install zsh-autosuggestions plugin into plugins folder

#### tmux 

- Install TPM to tmux so that other plugins will load
- Install fzf

#### After that

- `install.sh` will place soft links to correct places (if it works?)
