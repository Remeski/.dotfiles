## Remeski dotfiles -- STOW

### Requirements

- a Nerdfont installed
- oh-my-zsh
- gnu stow

### Installing

#### oh-my-zsh

- Install zsh-autosuggestions plugin into plugins folder

#### tmux 

- Install TPM to tmux so that other plugins will load
- Install fzf

### Using STOW

- run `stow \[target\]` to install symlinks to correct places
