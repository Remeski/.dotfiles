# Hyprland

## Requirements
- `hyprland`
- `swww` (older `hyprpaper`)
- `hyprlock`
- `hypridle`
- `hyprshot`
- `bluetui`
- `pulsemixer`
- `elephant` and `walker` (older `rofi`)
- `kitty`
- `matugen`

## Matugen

For gtk theme switching create `gtk.css` files in `~/.config/gtk-X.0/` that has

```css
@import 'colors.css';
```
